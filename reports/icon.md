🎨 Pastel Color Palette

Primary (brand)

Lavender — #CDB4DB

Secondary (supporting tones)

Mint Green — #B8E6C1
Peach — #FFD6A5

Accent (for highlights / CTAs)

Soft Coral — #FFB4A2

Background (light mode)

Warm Off-White — #FAFAF7

Text / Contrast (soft dark)

Muted Navy — #3D4A5A

🧠 How to use it (quick guidance)
Blue + Lavender → main branding / headers
Mint + Peach → sections, cards, UI blocks
Coral → buttons / important actions only (don’t overuse)
Off-white → clean UI background
Muted Navy → keeps readability without harsh black

Pastel palettes can feel “washed out” quickly, so always:

pair them with enough whitespace
use slightly darker text than you think you need
keep only 1 strong accent color active at a time
